Getting started with NervaOS

====================

![NervaOS](https://github.com/NervaOS/manifest/blob/master/banner.png?raw=true)

Initialize Local Repository

-------------

```bash

  repo init -u https://github.com/NervaOS/manifest -b <select_version>

```

Syncing Repository

-------------

```bash

  repo sync

```

Lunch Command

-------------

```bash

  . build/envsetup.sh

```
